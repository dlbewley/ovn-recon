# Contributing to OVN Recon

We'd love your help!

## Issues

Please use [GitHub Issues](https://github.com/dlbewley/ovn-recon/issues) to report bugs or request features.

## Pull Requests

1.  Fork the repository.
2.  Create a branch for your feature or fix.
3.  Commit your changes.
    *   Please follow the [Conventional Commits](https://www.conventionalcommits.org/) specification.
4.  Push to your fork.
5.  Submit a Pull Request.

## Development

See [docs/BUILDING.md](docs/BUILDING.md) for instructions on how to set up your development environment.

## Code Style

-   Run `npm run lint` to ensure your code matches the project's style.
-   Run `npm run test` to ensure tests pass.
